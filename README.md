# Investment Analyst Bot (Python)

Бот для анализа криптовалют в стиле профессионального аналитика с поддержкой **любой монеты Binance**, если для нее доступна торговая пара на бирже.

## Что умеет
- Загружает историю цен из Binance API (без API-ключа).
- Понимает прямой запрос пользователя, например `--query "проанализируй pepe"`.
- Автоматически пытается подобрать правильную пару Binance через `exchangeInfo` и переводит запрос в формат вроде `PEPEUSDT`.
- Считает ключевые метрики:
  - доходность за период;
  - волатильность (годовая);
  - VaR 95%;
  - максимальная просадка;
  - тренд (SMA20/SMA50);
  - импульс за 14 дней;
  - положение цены в 30-дневном временном цикле.
- Формирует рекомендации:
  - `BUY` / `HOLD` / `AVOID`;
  - риск-профиль: `LOW` / `MEDIUM` / `HIGH`.
- Генерирует подробный текстовый отчет по каждому инструменту и сводную таблицу.
- Может добавлять **AI / Strategy Blueprint**:
  - если задан `OPENAI_API_KEY`, делает запрос в OpenAI-compatible Chat Completions API и просит LLM собрать алгоритм анализа;
  - если ключа нет, строит детерминированный адаптивный алгоритм на основе тренда, риска, импульса и временных циклов.

## Установка
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск
Обычный анализ по списку тикеров:
```bash
python -m bot_analyzer.main --symbols BTCUSDT ETHUSDT SOLUSDT --period 6mo
```

Прямой запрос на монету Binance:
```bash
python -m bot_analyzer.main --query "проанализируй pepe" --period 3mo
```

Анализ с генерацией алгоритма/blueprint:
```bash
python -m bot_analyzer.main --query "analyse doge" --period 6mo --with-ai
```

Пример сохранения отчета в файл:
```bash
python -m bot_analyzer.main --symbols BTCUSDT ETH-USD --period 1y --with-ai --output report.txt
```

## Параметры
- `--symbols` список символов Binance или названий монет.
- `--query` прямой запрос на анализ одной монеты; из запроса извлекается последний значимый токен.
- `--period` период: `1mo`, `3mo`, `6mo`, `1y`, `2y`, `5y`.
- `--output` путь к файлу для сохранения отчета.
- `--with-ai` добавить AI / Strategy Blueprint.

## Переменные окружения для LLM
- `OPENAI_API_KEY` — если задан, включается запрос к OpenAI-compatible API.
- `OPENAI_BASE_URL` — опционально, если нужен совместимый endpoint.
- `OPENAI_MODEL` — опционально, модель для генерации стратегии (по умолчанию `gpt-4o-mini`).

## Примечания
- Используйте торговые пары Binance, например `BTCUSDT`, `ETHUSDT`, `SOLUSDT`.
- Форматы `BTC-USD`, `BTC/USD`, `BTCUSD`, а также простые запросы вроде `btc`, `pepe`, `doge` автоматически нормализуются и сопоставляются с Binance-парой.
- Если по монете существует несколько подходящих `USDT`-пар, бот выбирает наиболее короткий и очевидный матч по base asset.
- Инструмент не дает инвестиционных гарантий; это аналитическая поддержка принятия решений.
